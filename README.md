# e-ngx-xgplayer

基于Angular的[西瓜播放器](http://h5player.bytedance.com/)。

## Usage

1. Install

	```shell
	npm install --save e-ngx-xgplayer@latest
	```

2. Add the ENgxPrintModule

	```typescript
	import {ENgxXGPlayerModule} from "e-ngx-xgplayer";
	@NgModule({
	    imports: [
	        ENgxXGPlayerModule
	    ]
	})
	```

3. Use in Template

	```html
	```

4. Use in Component

	```typescript
	```
